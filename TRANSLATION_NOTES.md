# 首轮翻译说明

## 核心术语

| 英文 | 简体中文 |
|---|---|
| Library | 课程库 |
| Lecture | 课时 |
| Section | 章节 |
| Course Curriculum / Content | 课程目录 |
| Career Tracks | 学习路线 |
| Continue Watching | 继续观看 |
| Resume Lecture | 继续学习 |
| Resources | 课程资料 |
| Bookmarks | 书签 |
| History | 观看历史 |
| Stats | 学习统计 |
| In progress | 进行中 |
| Completed / Done | 已完成 |
| Not started | 未开始 |
| Up next | 接下来学习 |
| Sleep timer | 睡眠定时 |

## 翻译取向

- `Lecture` 统一译为“课时”，避免与课程 `Course` 混淆。
- `Career Track` 译为“学习路线”，更符合网课管理场景。
- `Resources` 译为“课程资料”，包含 PDF、代码、压缩包等附件。
- 播放器中的 `Content` 译为“目录”。
- `Resume` 根据场景译为“继续播放”或“继续学习”。

构建后应优先核对播放器、导入弹窗、设置页和课程详情页，这些区域包含较多动态文本。
